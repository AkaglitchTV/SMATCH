# Spécifications Techniques et Code Source — Application SMATCH (2026)

Ce document centralise l'architecture front-end de l'application **Smatch**, une plateforme de mise en relation de partenaires sportifs outdoor (mode Hiver ❄️ / Été ☀️). Il contient la structure HTML, le design system Tailwind CSS, et la logique JavaScript globale.

---

## 🗺️ Architecture des Fichiers

L'application est structurée autour des fichiers suivants :
1. `recherche.html` : Formulaire de recherche dynamique multi-critères.
2. `resultats.html` : Grille d'affichage des profils compatibles avec filtres.
3. `nav.js` : Gestion globale de la navigation et du switch de saison (Hiver/Été).
4. `cities.js` : Base de données des destinations et moteur d'autocomplétion.
5. `store.js` : Gestion de la persistance locale (`localStorage`).
6. `trip_data.js` : Configuration globale et data statique.
7. `mobile.css` : Ajustements et correctifs UI pour terminaux mobiles.

## 🛠️ Code Source

### 1. Structure Commune & Navigation (`nav.js`)
Gère l'injection du header et la bascule dynamique des thèmes Hiver (Cyan) / Été (Amber).

```javascript
document.addEventListener('DOMContentLoaded', () => {
  const navMount = document.getElementById('nav-mount');
  if (!navMount) return;

  const currentMode = localStorage.getItem('snm_mode') || 'hiver';

  navMount.innerHTML = `
    <header class="w-full bg-slate-950/80 border-b border-slate-900 sticky top-0 z-50 backdrop-blur-md">
      <div class="max-w-6xl mx-auto px-4 h-16 flex items-center justify-between">
        <a href="recherche.html" class="flex items-center gap-2 font-black text-xl tracking-tight text-white">
          🏔️ SMATCH
        </a>
        <div class="flex items-center gap-4">
          <div class="bg-slate-900 rounded-full p-1 flex items-center border border-slate-800">
            <button id="toggle-hiver" onclick="switchSeason('hiver')" class="px-3 py-1 rounded-full text-xs font-bold transition-all">❄️ Hiver</button>
            <button id="toggle-ete" onclick="switchSeason('ete')" class="px-3 py-1 rounded-full text-xs font-bold transition-all">☀️ Été</button>
          </div>
        </div>
      </div>
    </header>
  `;

  updateToggleState(currentMode);
});

function switchSeason(mode) {
  localStorage.setItem('snm_mode', mode);
  updateToggleState(mode);
  if (typeof onSeasonChange === 'function') {
    onSeasonChange(mode);
  } else {
    window.location.reload();
  }
}

function updateToggleState(mode) {
  const btnHiver = document.getElementById('toggle-hiver');
  const btnEte = document.getElementById('toggle-ete');
  if (!btnHiver || !btnEte) return;

  if (mode === 'ete') {
    btnEte.className = "px-3 py-1 rounded-full text-xs font-bold bg-amber-500 text-slate-950 shadow-md";
    btnHiver.className = "px-3 py-1 rounded-full text-xs font-bold text-slate-500 hover:text-slate-300";
  } else {
    btnHiver.className = "px-3 py-1 rounded-full text-xs font-bold bg-cyan-500 text-slate-950 shadow-md";
    btnEte.className = "px-3 py-1 rounded-full text-xs font-bold text-slate-500 hover:text-slate-300";
  }
}
